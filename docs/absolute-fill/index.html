<AbsoluteFill />

A helper component - it is an absolutely positioned <div> with the following styles:

css
.absolute-fill {
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  width: 100%;
  height: 100%;
  display: flex;
  flex-direction: column;
}
